System Environment:

1. Download and install Java JDK 8

2. Download and install Apache Spark
	Spark 2.4.4
	Apache Hadoop 2.7

3. Download and install Python 3.7

4. Install Required Libraries.


Build and Run Steps:

1. Keep train and test data in same directory with program files

2. Keep main.py, LinearR.py, DecisionTree.py, RidgeRegression.py, RandomForest.py in the same directory

2. Run the main.py file.

Output:

Linear Regression Model. RMSE 1130  
Ridge Regression Model. RMSE 1059
Decision Tree Model. RMSE 1073
Random Forrest Model. RMSE 1068
